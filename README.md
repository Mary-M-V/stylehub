## Що ще потрібно зробити

### 🔴 Критично
1. Активувати GA4 або GTM — знайти блок `<!-- ═══ ANALYTICS ═══ -->` в index.html

### 🟡 Важливо
2. Створити favicon.png (32×32px) → завантажити в /img/
3. Підключити до Google Search Console
4. Створити sitemap.xml та robots.txt
5. Перевірити OG-превью: opengraph.xyz
